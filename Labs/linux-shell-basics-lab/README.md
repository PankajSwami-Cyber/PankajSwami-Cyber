# Linux Shell Basics Lab

## Overview

This lab focused on learning how to interact with the Linux operating system through the Bash shell. As a cybersecurity professional, understanding shell commands is essential for system administration, troubleshooting, automation, and security operations.

The objective of this lab was to explore how the shell receives user input, processes commands, and returns output.

---

## Lab Objectives

* Generate output using the `echo` command
* Perform basic arithmetic calculations using the `expr` command
* Clear the terminal screen using the `clear` command
* Explore shell input and output behavior

---

## Scenario

In this lab, I practiced communicating with the operating system through the Bash shell. I entered commands and observed how the shell interpreted the input and displayed the resulting output.

The exercises included generating text output, performing mathematical calculations, and managing the terminal display.

---

## Commands Used

### 1. Generate Output

```bash
echo hello
```

**Output:**

```text
hello
```

---

### 2. Generate Output Using Quotation Marks

```bash
echo "hello"
```

**Output:**

```text
hello
```

---

### 3. Display Multiple Words

```bash
echo Pankaj Swami
```

**Output:**

```text
Pankaj Swami
```

---

### 4. Perform a Subtraction Calculation

```bash
expr 32 - 8
```

**Output:**

```text
24
```

---

### 5. Perform a Multiplication Calculation

```bash
expr 3500 \* 12
```

**Output:**

```text
42000
```

> Note: The asterisk (*) must be escaped with a backslash (`\`) because it has a special meaning in the shell.

---

### 6. Clear the Terminal Screen

```bash
clear
```

This command clears all visible content from the terminal window.

---

## Key Concepts Learned

* The Bash shell accepts commands as user input.
* Commands are processed by the operating system and returned as output.
* The `echo` command displays text in the terminal.
* The `expr` command evaluates mathematical expressions.
* The `clear` command improves terminal readability by removing previous output from view.

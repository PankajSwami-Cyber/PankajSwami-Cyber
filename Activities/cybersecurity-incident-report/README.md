# Cybersecurity Incident Report

## Project Overview
This project analyzes a network intrusion incident involving a TCP SYN Flood Denial of Service (DoS) attack.

## Skills Demonstrated
- Network Traffic Analysis
- Wireshark Log Investigation
- TCP Three-Way Handshake Analysis
- Incident Reporting
- DoS Attack Identification

## Attack Identified
TCP SYN Flood (DoS)

## Tools Used
- Wireshark
- TCP/IP Protocol Analysis
- Cybersecurity Incident Reporting

## Key Findings
The attacker flooded the web server with SYN packets, exhausting server resources and preventing legitimate users from accessing the website.

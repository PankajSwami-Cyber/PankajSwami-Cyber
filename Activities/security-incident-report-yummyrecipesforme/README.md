# Security Incident Report

## Project Overview
This project investigates a security incident involving a compromised website (yummyrecipesforme.com).

## Skills Demonstrated
- Network traffic analysis
- TCP/IP model
- DNS analysis
- HTTP protocol investigation
- Malware detection
- Incident response
- Security documentation

## Findings
- HTTP was used to deliver malicious content.
- DNS requests redirected users to a malicious website.
- Website compromise occurred through a brute-force attack against an administrator account.

## Recommendation
Implement Two-Factor Authentication (2FA) to reduce the risk of unauthorized access.

## Tools Used
- tcpdump
- DNS Analysis
- HTTP Traffic Inspection
- Sandbox Environment

# Incident Report Analysis Using NIST CSF

## Overview
This project analyzes a Denial of Service (DoS) attack against a multimedia company using the NIST Cybersecurity Framework (CSF).

## Scenario
A malicious actor launched an ICMP flood attack through an unconfigured firewall, causing network services to become unavailable for two hours.

## NIST CSF Functions Applied

- Identify
- Protect
- Detect
- Respond
- Recover

## Skills Demonstrated

- Incident Response
- Risk Assessment
- Network Security
- Firewall Security
- IDS/IPS
- NIST Cybersecurity Framework
